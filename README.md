# SPI Communication Between STM32 and Arduino UNO

This project demonstrates **SPI communication using bare-metal (register-level) programming** on the **STM32F446RE (Nucleo-64)** with an **Arduino UNO** acting as the SPI Slave.

The **STM32F446RE** is configured as the **SPI Master**, and the **Arduino UNO** responds as a **Slave**

##  Hardware Used
1. **STM32 Nucleo-F446RE**
2. **Arduino UNO**
3. **USB Logic Anaylyzer**
4. Jumper wires  
5. USB cables

##  Software Used

1. **STM32 Cube IDE**
2. **Ardiuno IDE**
3. **Saleae USB logic analyzer software**

## **SPI Configuration**
1. **STM32F446RE NUCLEO** Configured as **SPI Master**
2. **Ardiuno UNO** Configured as **SPI Slave**
